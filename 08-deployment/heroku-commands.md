# Heroku commands

// Login
```
heroku login
```

// Get auth token
```
heroku auth:token
```

// List my apps
```
heroku apps
```

// Run bash on the instance
```
heroku run bash --app nodejs-nights-development
```

// Get instance logs 
```
heroku logs --tail --app nodejs-nights-development
```

// Disonnect from terminal
CTRL+D - disconnect
